Detailed Description

A simple and fast stack object for doing memory management live marking. This has minimal overrun protections, so the using code needs to perform appropriate "stack is full" checks.

◆ ensureSpace()

LiveStack * LiveStack::ensureSpace ( size_t  needed)

Ensure that the live stack is going to be large enough to fit all of the objects that might be marked from a large collection. This allows us to proactively expand the stack before a GC event is taking place, and potentially raise a real Rexx EOM condition before we're at a critical point.

Parameters
neededThe number of live stack slots this object might require. It is assumed that the stack needs to be expanded at this point.
Returns
A newly allocated stack with the entries from this stack copied over to it.
