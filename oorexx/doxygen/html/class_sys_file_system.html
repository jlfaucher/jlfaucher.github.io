test if an individual file is a case sensitive name

Most Unix file systems by default are case-sensitive (with the exception of e. g. Darwin). But since Linux kernel 5.2 a per-directory case-insensitivity is supported on ext4 file systems. On Darwin, partitions may be configured to be case sensitive.

Returns
true if the given file is located within a case-sensitive directory.

Test if a file or directory is case-sensitive.

On Windows, case-sensitivity is defined on a per-directory basis. Once a directory has been set as case-sensitive, files in this directory and any newly created subdirectories within this directory become case-sensitive. Existing subdirectories and files within those existing subdirectories stay as they are.

Returns
true if this is a case-sensitive directory, or a file within a case-sensitive directory, false otherwise.
