◆ initCache()

void RexxIntegerClass::initCache ( )

This method will pre-allocate 111 integer objects, -10 .. 100. These will then be used whenever a request for an integer between -10 and 100 is requested this should help reduce some of our memory requirements and trips through memory_new.
